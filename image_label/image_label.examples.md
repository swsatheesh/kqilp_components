#### Basic
```
<ImageLabel altText='testimage'>
    My Label
</ImageLabel>
```

#### Custom src
```
<ImageLabel
    src='http://placeimg.com/320/240/nature'
    altText='testimage'
>
    My Label
</ImageLabel>
```

#### Without label
```
<ImageLabel src='http://placeimg.com/320/240/nature' />
```
#### Adding custom class for image and label
```
<ImageLabel
    src='http://placeimg.com/320/240/nature'
    altText='testimage'
    imgClass={'image-class'}
    labelClass={'label-class'}
>
    My Label
</ImageLabel>
```
